GitHub Notifier

To check how many unread notifications you have, this Chrome extension requires a GitHub personal access token.

To generate a new GitHub access token:

  1. Go to this page.
  2. Make sure to check the checkbox for the notifications scope. You don't need to give this extension access to anything else.
  3. Enter a description and generate your token.

Then copy the token from GitHub and enter it below:
